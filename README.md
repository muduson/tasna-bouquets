# Tasna Bouquets - Floral Shop Website

A beautiful one-page website for Tasna Bouquets, a floral shop showcasing bouquets with prices and direct WhatsApp ordering.

## Features
- Responsive design that works on all devices
- Bouquet gallery with images and prices
- Direct WhatsApp API integration for instant ordering
- Service showcase section
- Smooth navigation

## How to Use
1. Open `index.html` in any browser
2. Click "Order via WhatsApp" on any bouquet to chat directly with the supplier
3. Update the supplier's phone number in the JavaScript section

## Customization
- Change supplier WhatsApp number: Edit `SUPPLIER_PHONE` variable in the script
- Add/remove bouquets: Modify the gallery section
- Update colors: Modify CSS variables

## Technologies Used
- HTML5
- CSS3 (Flexbox, Grid)
- JavaScript
- Font Awesome Icons
- WhatsApp API (wa.me)
